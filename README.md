## 🚀 Featured Project

### 🍽️ Kenyan Meal Planner
> A Web-Based Meal Planning and Budgeting System for Kenyan Homes

**🔗 [View Project](https://github.com/emmamuriuki1-source/kenyan-meal-planner)**

---

### 📌 Problem Statement

Kenyan households struggle to plan affordable, nutritious meals due to
fluctuating market prices, busy schedules, and lack of integrated digital
tools. Existing platforms such as KenyanPlate and Budget Yangu address
either meal planning or budgeting separately, but none combine both in a
locally relevant, interactive system. This forces families to overspend,
waste food, and rely on repetitive or unhealthy meals.

This system was developed to bridge that gap — providing a single platform
where Kenyan households can plan weekly meals using local recipes, enter
current market prices, calculate grocery costs in real time, and manage
their food budget effectively.

---

### ✅ Key Features

- Weekly meal planning with authentic Kenyan recipes (Ugali, Githeri, Pilau, Chapati and more)
- Real-time grocery cost calculation per ingredient
- Budget tracking with colour-coded progress bar
- Budget-saving ingredient substitution suggestions
- Nutritional summary — Vegetables, Fruits, Dairy, Protein, Carbohydrates
- Downloadable weekly budget and nutrition report (PDF)
- Admin panel — user management, recipe management, system analytics
