## 🚀 Featured Project

### 🍽️ Kenyan Meal Planner
> A Web-Based Meal Planning and Budgeting System for Kenyan Homes

A full-stack web application that helps Kenyan households plan weekly meals,
track grocery costs, manage food budgets, and maintain nutritional balance.

**🔗 [View Project](https://github.com/emmamuriuki1-source/kenyan-meal-planner)**

**Key Features:**
- ✅ Weekly meal planning with authentic Kenyan recipes
- ✅ Real-time grocery cost calculation per ingredient
- ✅ Budget tracking with colour-coded progress bar
- ✅ Budget-saving ingredient substitution suggestions
- ✅ Nutritional summary (Veg, Fruit, Dairy, Protein, Carbs)
- ✅ Downloadable weekly budget and nutrition report
- ✅ Admin panel — user management, recipe management, analytics

**Tech Stack:**
`PHP` `MySQL` `JavaScript` `HTML5` `CSS3` `Chart.js` `XAMPP` `AJAX`
